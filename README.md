--Cannabis in Canada - Impacts of Legalisation and Impact on the Canadian Economy--


The purpose of this project is to:

Evaluate and project the economic impact of the cannabis industry in Canada, post legalisation (2017)
Create a predictive model which aims to identify economic impact going forward

Benefits

Understand contribution of cannabis industry to Canadian economy
Understand current and future usage trends
Evaluate potential impacts in the future
Evaluate if legalisation has led to increased usage (as per hypothesis)
